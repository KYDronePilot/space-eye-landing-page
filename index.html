SpaceEye

Free

Live satellite imagery for your Mac or Windows desktop background.

All installation methods

Your desktop background updates throughout the day

New images are downloaded every 20 minutes, giving an up-to-date view of the Earth on your desktop.

12 views from 5 satellites

There are currently 12 views from 5 different weather satellites: Himawari-8, GOES-17 (West), GOES-16 (East), Meteosat-8, and Meteosat-11.

High resolution imagery

Satellite views are downloaded in up to 8K resolution for crystal clear views of the Earth.

Cross-platform

Can be installed on both macOS and Windows.

Install on Mac

Download from the Mac App Store

Install on Windows

Recommended: Download from the Microsoft Store

or follow the installation instructions to download and run the installer from the latest GitHub release.


Note: You will have to dismiss warnings about the download being potentially dangerous with this method.
